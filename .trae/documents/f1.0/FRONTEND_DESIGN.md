> 视频分析工作台前端设计文档

## 1. 设计概述

### 1.1 设计决策

| 维度 | 选择 | 说明 |
|------|------|------|
| UI风格 | 科技感型 | 深色主题，动画效果丰富 |
| 布局 | 侧边栏导航 + 主内容区 | 左导航，右内容 |
| 组件库 | Tailwind CSS | 快速开发，自建组件 |
| 播放器 | video.js | 通用视频播放 |
| 响应式 | 仅桌面端 | 简化开发 |
| 实时更新 | SSE | 后端已支持 |

### 1.2 技术栈

```json
{

  "http": "Axios",
  "styling": "Tailwind CSS 3.3",
  "icons": "Lucide React",

}
```

## 2. UI/UX 设计规范

### 2.1 色彩系统

| 变量名 | 色值 | 用途 |
|--------|------|------|
| `app-bg` | `#0a0e17` | 主背景 |
| `app-bg-secondary` | `#111827` | 次级背景/侧边栏 |
| `app-card` | `#1f2937` | 卡片背景 |
| `app-primary` | `#3b82f6` | 主色调/按钮 |
| `app-success` | `#10b981` | 成功状态 |
| `app-warning` | `#f59e0b` | 警告状态 |
| `app-error` | `#ef4444` | 错误状态 |
| `app-text` | `#f9fafb` | 主文字 |
| `app-text-secondary` | `#9ca3af` | 次级文字 |

### 2.2 字体

- 主字体: `-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial, sans-serif`
- 代码: 系统等宽字体

### 2.3 间距系统

- 页面内边距: `24px` (`p-6`)
- 卡片间距: `16px` (`gap-4`)
- 组件间距: `12px` (`space-y-3`)

### 2.4 动画规范

| 动效 | 实现 | 参数 |
|------|------|------|
| 页面切换 | 淡入 + 上移 | `150ms` |
| 按钮 hover | 发光效果 | `box-shadow` |
| 卡片 hover | 上浮 + 阴影 | `translateY(-4px)` |
| 加载骨架屏 | shimmer 动画 | `1.5s infinite` |

## 4. 页面设计

### 4.1 视频上传页 (`/videos`)

#### 布局

```
┌─────────────────────────────────────────────────────────────┐
│  视频管理                                      [刷新列表]   │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────┐           │
│  │              上传区域 (虚线边框)              │           │
│  │         点击上传视频 或 拖拽文件到此处        │           │
│  │           支持 MP4, AVI, MOV, MKV           │           │
│  └─────────────────────────────────────────────┘           │
│                                                             │
│  ─── 已上传视频 ─────────────────────────────────         │
│  ┌────────┐ ┌────────┐ ┌────────┐ ┌────────┐              │
│  │ 缩略图 │ │ 缩略图 │ │ 缩略图 │ │ 缩略图 │              │
│  │ 名称   │ │ 名称   │ │ 名称   │ │ 名称   │              │
│  │ 时长   │ │ 时长   │ │ 时长   │ │ 时长   │              │
│  └────────┘ └────────┘ └────────┘ └────────┘              │
└─────────────────────────────────────────────────────────────┘
```

#### 交互

| 交互 | 行为 |
|------|------|
| 点击上传区 | 打开文件选择器 |
| 拖拽文件 | 高亮边框 + 放置动画 |
| 视频卡片 hover | 显示删除按钮 |
| 点击删除 | 确认删除，更新列表 |

## 6. API 接口

### 6.1 视频接口

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/v1/videos/upload` | 上传视频 |
| GET | `/api/v1/videos` | 获取视频列表 |
| GET | `/api/v1/videos/{id}` | 获取视频详情 |
| DELETE | `/api/v1/videos/{id}` | 删除视频 |

### 6.2 任务接口

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/v1/tasks` | 创建任务 |
| GET | `/api/v1/tasks` | 获取任务列表 |
| GET | `/api/v1/tasks/{id}` | 获取任务详情 |
| POST | `/api/v1/tasks/{id}/start` | 启动任务 |
| POST | `/api/v1/tasks/{id}/stop` | 停止任务 |
| DELETE | `/api/v1/tasks/{id}` | 删除任务 |
| GET | `/api/v1/tasks/{id}/output` | 获取输出地址 |

### 6.3 模型接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/v1/models` | 获取模型列表 |
| GET | `/api/v1/models/{name}` | 获取模型详情 |
| GET | `/api/v1/models/instances` | 获取实例状态 |
