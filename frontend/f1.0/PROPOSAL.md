# f1.0 前端方案设计

## 1. 设计概述

基于 v1.0 API 规格和用户选择：
- UI风格: 科技感型（深色主题 + 动画效果）
- 布局: 侧边栏导航 + 主内容区
- 优先级: 视频上传 > 任务创建 > 模型管理 > 任务监控

---

## 2. UI/UX 设计

### 2.1 整体风格

**配色方案**:
- 主背景: `#0a0e17` (深邃蓝黑)
- 次背景: `#111827` (深灰蓝)
- 卡片背景: `#1f2937` (灰蓝)
- 主色调: `#3b82f6` (科技蓝)
- 强调色: `#10b981` (成功绿)
- 警告色: `#f59e0b` (警告橙)
- 错误色: `#ef4444` (错误红)
- 文字主色: `#f9fafb`
- 文字次色: `#9ca3af`

**动效**:
- 页面切换: 淡入 + 轻微上移动画 (150ms)
- 按钮 hover: 发光效果
- 卡片 hover: 轻微上浮 + 阴影加深
- 数据加载: 骨架屏 shimmer 动画

### 2.2 布局结构

```
┌─────────────────────────────────────────────────────────────┐
│  顶部栏 (64px)                                        │
│  Logo | 搜索框 | 用户头像                           │
├──────────┬──────────────────────────────────────────┤
│          │                                          │
│  侧边栏  │         主内容区                         │
│  (240px) │                                        │
│          │    ┌─────────────────────────────┐         │
│  导航项  │    │  页面标题 + 操作按钮   │         │
│          │    ├─────────────────────────────┤         │
│  - 上传  │    │                     │         │
│  - 任务  │    │     内容区域         │         │
│  - 模型  │    │                     │         │
```

### 2.3 响应式断点

| 断点 | 宽度 | 侧边栏 |
|------|------|--------|
| Desktop | ≥1024px | 展开 240px |
| Tablet | 768-1023px | 折叠 64px |
| Mobile | <768px | 隐藏 / 底部导航 |

## 5. 技术栈

- **框架**: React 18 + TypeScript
- **构建**: Vite
- **状态**: React Context + useReducer
- **路由**: React Router 6
- **UI**: Tailwind CSS 或自建组件库
- **HTTP**: Axios
- **图标**: Lucide React
